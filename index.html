Local Storage
  • Browser এর মধ্যে Data স্থায়ীভাবে store করে রাখতে পারে
  • Browser বন্ধ করে দিলেও সকল Data store এর মধ্যে থেকে যায়
  • এটি 5 থেকে 10 mb এর মত data store করে রাখতে পারে
  • Local storage এর data javascript বা সরাসরি delete করা যায়
Session Storage
  • Browser এর মধ্যে Data স্থায়ীভাবে store করে রাখতে পারে না
  • Browser বন্ধ করে দিলে সকল Data store এর মধ্যে থেকে চলে যায়
  • ক্রোম এবং ফায়ারফক্স সহ বেশিরভাগ ব্রাউজার সেশন স্টোরেজে প্রায় 10 এমবি ডেটা store করে রাখতে পারে।
  • session storage এর data javascript বা সরাসরি delete করা যায়
